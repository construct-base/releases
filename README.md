# Construct - Releases

Official releases repository for Construct.

## Download Latest Version

Visit the [Releases](https://github.com/construct-base/releases/releases) page to download the latest version of Construct.

## Supported Platforms

- macOS (Apple Silicon)
- macOS (Intel)
- Windows
- Linux

## Report Issues

If you encounter any bugs or have feature requests, please [open an issue](https://github.com/construct-base/releases/issues).

## About Construct

Construct is a privacy-focused browser built on Firefox with enhanced features and customization options.

## Update System

Construct automatically checks for updates from https://one.base.al/
